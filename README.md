# 🎉 팬 커뮤니티 웹 퍼블리싱 프로젝트

## 📌 프로젝트 개요

해당 프로젝트는 **팬 커뮤니티 웹사이트를 위한 퍼블리싱 작업**으로,
모바일 우선(Mobile-First) 기반의 반응형 구조와 직관적인 UI/UX를 중심으로 개발되었습니다.

커뮤니티 특성상 **게시글·댓글 상호작용**, **좋아요 기능**, **토글 메뉴**,
**이미지/텍스트 기반 인터페이스**, **경량화된 JS 기반 인터랙션**이 핵심 요소입니다.

> ⚠️ 참고
> 본 프로젝트는 회사 내부 지연으로 인해 운영 서버에는 최신 작업물이 반영되지 않았으며,
> 본 저장소에 포함된 코드는 **완성된 로컬 퍼블리싱 작업물**입니다.
> 실제 운영중인 사이트 : https://veeva.fan/t1

---

## 🗓️ 개발 기간

**2025.8 – 2025.10 (약 3개월, 프리랜서 외주 작업)**

* 재택 기반 원격 협업(화상회의 포함)
* 기획&디자인/백엔드 팀과 요구사항 조율

---

## 🎨 컨셉 & 디자인 방향

* **모바일 우선(Mobile-First)**

  * 실제 유입의 90% 이상이 모바일 사용자인 점을 고려한 설계
* **가독성 중심의 커뮤니티 UI**

  * 게시글·댓글 구조 명확화
  * 좋아요/댓글 버튼 직관적 배치
* **심플하고 빠른 인터랙션**

  * 불필요한 애니메이션 최소화
  * 사용자 흐름 방해 없이 자연스러운 전환

---

## ✨ 구현 기능

### 📍 1. 게시글 & 댓글 UI 퍼블리싱

* 반응형 게시글 카드
* 작성자 정보, 시간, 카운트 노출
* 이미지 첨부 게시글 UI 구성
* 댓글 depth(1레벨) 기반 구조화

---

### 📱 2. 반응형 레이아웃

* 375px 기반 모바일 최적화
* PC 대응
* Flexbox + CSS Custom Properties 적용
* 이미지 영역 가변 비율 관리

---

### 📂 3. 공통 UI 구성

* 상단 네비게이션
* 뒤로가기/홈 버튼
* 사용자 프로필 이미지 사이드바
* 리스트·디테일 페이지 UI 구성

---
